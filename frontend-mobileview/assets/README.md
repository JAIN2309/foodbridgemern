# Assets Folder

Place your app assets here:

- `icon.png` - App icon (1024x1024)
- `splash.png` - Splash screen (1242x2436)
- `adaptive-icon.png` - Android adaptive icon (1024x1024)
- `favicon.png` - Web favicon (48x48)

## Temporary Placeholders

For now, you can use placeholder images or generate them using:
- https://www.figma.com
- https://www.canva.com
- Or any image editor

## Recommended Sizes

- **icon.png**: 1024x1024 px
- **splash.png**: 1242x2436 px (iPhone 11 Pro Max)
- **adaptive-icon.png**: 1024x1024 px
- **favicon.png**: 48x48 px

The app will work without these assets, but you'll see warnings in the console.
